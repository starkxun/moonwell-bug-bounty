# MIP-X42: Anthias Labs Monthly Recommendations (02/2026)

## Base

## Summary

This proposal includes a structural change to Moonwell’s allocator framework:

- The public allocator for vaults is being removed. No longer will public
  allocator calls be permitted.

- Block Analitica is being removed as an allocator and replaced with Anthias.

### Risk Parameters

A proposal to adjust the following risk parameters:

| Parameters             | Current Value | Recommended Value |
| ---------------------- | ------------- | ----------------- |
| LBTC Collateral Factor | 84%           | 83%               |
| tBTC Collateral Factor | 82%           | 81%               |
| WELL Borrow Cap        | 22M           | 34M               |

<sub> \*Cap changes will be implemented via Cap Guardian </sub>
