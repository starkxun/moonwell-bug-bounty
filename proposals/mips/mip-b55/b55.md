# MIP-B55: MetaMorpho Vault Incentive Extension

### **Summary**

This proposal extends WELL token incentives for four MetaMorpho vaults on Base
(USDC, WETH, EURC, and cbBTC) from January 16th to February 2nd, 2026. The
meUSDC vault is excluded from this extension. Reward rates are maintained at the
same APY as the previous MIP-X40 campaign.

### **Background**

MIP-X40 established 14-day vault incentive campaigns for MetaMorpho vaults on
Base, scheduled to end on January 16th, 2026. To ensure continued
incentivization without interruption, this proposal creates follow-up campaigns
that begin exactly when X40 vault campaigns conclude.

The campaigns maintain the same reward rate (tokens per second) as X40, ensuring
depositors continue to receive consistent APY yields.

### **Proposal**

This proposal creates Merkle incentive campaigns for the following vaults:

| Vault            | Address                                    | WELL Amount         |
| ---------------- | ------------------------------------------ | ------------------- |
| USDC MetaMorpho  | 0xc1256Ae5FF1cf2719D4937adb3bbCCab2E00A2Ca | ~1,356,325 WELL     |
| WETH MetaMorpho  | 0xa0E430870c4604CcfC7B38Ca7845B1FF653D0ff1 | ~454,279 WELL       |
| EURC MetaMorpho  | 0xf24608E0CCb972b0b0f4A6446a0BBf58c701a026 | ~271,050 WELL       |
| cbBTC MetaMorpho | 0x543257eF2161176D7C8cD90BA65C2d4CaEF5a796 | ~110,946 WELL       |
| **Total**        |                                            | **~2,192,600 WELL** |

**Campaign Timing:**

- Start: January 16, 2026 13:30:18 UTC (when X40 vaults end)
- End: February 2, 2026 00:00:00 UTC
- Duration: ~16.44 days (1,420,182 seconds)

**Amount Calculation:** To maintain the same APY as X40, amounts are scaled
proportionally:

```
new_amount = x40_amount * (new_duration / x40_duration)
new_amount = x40_amount * (1,420,182 / 1,209,600)
new_amount = x40_amount * 1.174
```

### **Rationale**

Continuing vault incentives ensures:

1. Uninterrupted rewards for MetaMorpho vault depositors
2. Sustained liquidity in Moonwell's vault ecosystem
3. Competitive yields compared to alternative DeFi protocols

The meUSDC vault is excluded as it operates under separate incentive structures.

### **Voting Options**

- **For:** Approve extension of vault incentives through February 2nd
- **Against:** Do not extend vault incentives
- **Abstain:** No preference
